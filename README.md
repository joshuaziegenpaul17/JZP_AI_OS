# 🚀 JZP_AI OS

🌐 Live Demo  
https://jzpaios.streamlit.app/

JZP_AI OS is a futuristic AI platform built using Streamlit, Groq AI, and Gemini Vision API.  
It combines multiple AI-powered tools into one modern dashboard application.

- 🤖 AI Chat Assistant

- 📄 PDF Summarizer
- 📑 Resume Analyzer

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/JZP_AI_OS.git
cd JZP_AI_OS
```

Install requirements:

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
GEMINI_API_KEY=your_gemini_api_key
```

---

# ▶️ Run Application

```bash
python -m streamlit run app.py
```

# 📂 Features Overview

## 💬 AI Chat
